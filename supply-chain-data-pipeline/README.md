# Supply Chain Data Pipeline & Analytics Platform

## Overview

This project builds an end-to-end ETL pipeline to process supply chain data including inventory, procurement, and logistics.

## Tech Stack

Python, SQL, Apache Airflow (concept), Pandas

## Architecture

Data Source → ETL Pipeline → Processed Data → Analytics

## Features

* ETL pipeline (extract, transform, load)
* Data cleaning and transformation
* KPI generation (inventory, supplier performance)
* Structured output for analytics
